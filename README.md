## Description
Ce projet contient les algorithmes réalisés dans le cadre du TP.  
Les fichiers `.algo` sont situés dans le dossier `src/`.
Ce projet Algobox permet :
- de calculer la distance entre deux points saisis par l'utilisateur
- de déterminer le rayon d’un cercle à partir de son diamètre
- d'afficher le message "hello github"
- d'afficher le message "bonjour viktor"

## Comment ouvrir / exécuter les fichiers
1. Installer AlgoBox (ou utiliser une version portable).
2. Ouvrir AlgoBox.
3. Menu **Fichier → Ouvrir**.
4. Sélectionner le fichier `.algo` dans le dossier `src/`.
